üçÉ Dockerizing MongoDB for Scalable Document-Based Applications

MongoDB is a popular NoSQL document database that stores data in flexible, JSON-like documents. Perfect for applications requiring flexible schemas, horizontal scaling, and rapid development.

Set Up MongoDB with Docker Compose‚Äã

Create a file named docker-compose.yml

version: '3.8'

services:
  mongodb:
    image: mongo:7.0
    container_name: mongodb
    restart: unless-stopped
    ports:
      - "27017:27017"
    environment:
      MONGO_INITDB_ROOT_USERNAME: admin
      MONGO_INITDB_ROOT_PASSWORD: password123
      MONGO_INITDB_DATABASE: myapp
    volumes:
      - mongodb-data:/data/db
      - ./init-scripts:/docker-entrypoint-initdb.d
    healthcheck:
      test: ["CMD", "mongosh", "--eval", "db.adminCommand('ping')"]
      interval: 30s
      timeout: 10s
      retries: 3

  # Optional: Mongo Express for database management
  mongo-express:
    image: mongo-express:latest
    container_name: mongo-express
    restart: unless-stopped
    ports:
      - "8081:8081"
    environment:
      ME_CONFIG_MONGODB_ADMINUSERNAME: admin
      ME_CONFIG_MONGODB_ADMINPASSWORD: password123
      ME_CONFIG_MONGODB_URL: mongodb://admin:password123@mongodb:27017/
      ME_CONFIG_BASICAUTH: false
    depends_on:
      - mongodb

volumes:
  mongodb-data:

Start MongoDB:

docker compose up -d

Check if it's running:

docker ps

Connect to MongoDB‚Äã

Connect directly to the container:

docker exec -it mongodb mongosh -u admin -p password123

Basic MongoDB Operations‚Äã

Create a database and collection, then insert data:

// Switch to database
use myapp;

// Clean old data
db.users.drop();

// ---------------- CREATE ----------------

  {
    name: "John Doe",
    email: "john@example.com",
    age: 30,
    skills: ["Node.js", "MongoDB"]
  },
  {
    name: "Jane Smith",
    email: "jane@example.com",
    age: 25,
    skills: ["JavaScript", "Python"]
  },
  {
    name: "Bob Wilson",
    email: "bob@example.com",
    age: 35,
    department: "Engineering"
  }
]);
print("\n‚úÖ CREATE: Inserted Users");
printjson(db.users.find().toArray());

// ---------------- READ ----------------
